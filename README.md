# 💼 Employee Salary Prediction System  

An end-to-end **Machine Learning application** that predicts employee salary using advanced ML algorithms. The project demonstrates a complete pipeline from **data preprocessing to deployment** with an interactive web interface.

---

## 🚀 Live Demo  
👉 https://veekshithaadharasani-employee-salary-predictor-app-pir1al.streamlit.app/

---

## 📌 Project Overview  
This project builds a robust predictive model to estimate employee salary using structured data. It covers **data preprocessing, feature engineering, model comparison, and deployment** as a user-friendly web application.

---

## ✨ Key Features  
- End-to-end ML pipeline (preprocessing → training → evaluation → deployment)  
- Model comparison: Random Forest, XGBoost, Gradient Boosting, Neural Networks  
- High prediction performance (**R² ≈ 0.90**)  
- Interactive web app using **Streamlit**  
- Real-time salary prediction  

---

## 🧠 Machine Learning Approach  

### 🔹 Data Processing  
- Handling missing values and outliers  
- Encoding categorical variables  
- Feature scaling and transformation  

### 🔹 Models Used  
- Random Forest Regressor  
- Gradient Boosting  
- XGBoost  
- Neural Networks  

### 🔹 Evaluation Metric  
- **R² Score ≈ 0.90**  
- Model benchmarking and tuning  

---

## 🛠️ Tech Stack  

- **Language:** Python
